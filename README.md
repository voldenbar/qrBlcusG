# 项目简介

## 核心功能
- 管理员模块：用户管理、信息管理、个人中心
- 旅游网管理：景点分类管理、旅游路线管理、酒店管理
- 用户模块：用户注册、忘记密码、推荐景点、热门景点、旅游景点查询与预订
- 旅游路线查询与收藏、酒店查询与预订、交通工具查询与预订、个人收藏、个人预订、个人中心AI模型

## 适用场景
- 旅游行业企业
- 旅游规划与咨询服务机构
- 旅游平台运营方

## 技术特点
- 采用Spring Boot和Vue.js技术栈，前后端分离，便于开发和维护
- 基于AI的定制化管理，提高旅游方案个性化推荐效果
- 用户界面友好，操作便捷，提升用户体验
- 数据库管理规范，确保数据安全性和稳定性
